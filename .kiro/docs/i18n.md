# i18n System

## Overview

- **Default locale**: `th` (Thai) — served at `/`
- **Secondary locale**: `en` (English) — served at `/en/`
- **Routing**: Astro built-in i18n with `prefixDefaultLocale: false`
- **Content format**: Typed TypeScript objects (ไม่ใช่ JSON)

## Configuration

```ts
// src/i18n/config.ts
export const defaultLocale: Locale = 'th';
export const locales = ['th', 'en'] as const;
export const localeTags: Record<Locale, string> = { th: 'th-TH', en: 'en-US' };
export const localizedPath = (locale: Locale, path = ''): string => { ... };
```

`localizedPath('th', 'privacy-policy')` → `/privacy-policy`
`localizedPath('en', 'privacy-policy')` → `/en/privacy-policy`

## Content File Structure

```
src/i18n/
├── types.ts                    ← Type definitions (source of truth)
├── config.ts                   ← Locale helpers
└── locales/
    ├── th/
    │   ├── common.ts           ← Header, footer, cookie banner
    │   ├── landing.ts          ← All landing sections content
    │   └── legal.ts            ← Privacy policy + terms
    └── en/
        ├── common.ts
        ├── landing.ts
        └── legal.ts
```

## Type Definitions (src/i18n/types.ts)

| Type | Used by | Contains |
|------|---------|----------|
| `Locale` | Everywhere | `'th' \| 'en'` |
| `SeoContent` | BaseLayout | title, description, OG title/description |
| `CommonContent` | Header, Footer, CookieBanner | brand, nav items, footer text, cookie text |
| `LandingContent` | LandingPage + sections | All 11 section contents + SEO |
| `LegalPageContent` | LegalPage | title, updated date, intro, sections array |
| `CardContent` | CardGridSection | `{ icon, title, description }` |
| `NavItem` | Header | `{ label, href }` |

## Pattern

1. **Page** imports content from locale file
2. **Page** passes content as typed props to layout/components
3. **Components** never import locale files directly — always receive via props

```astro
---
// src/pages/index.astro
import { commonTh } from '../i18n/locales/th/common';
import { landingTh } from '../i18n/locales/th/landing';
---
<BaseLayout locale="th" seo={landingTh.seo} common={commonTh}>
  <LandingPage content={landingTh} lineUrl={lineUrl} />
</BaseLayout>
```

## Adding/Editing Content

1. แก้ type ใน `src/i18n/types.ts` (ถ้าโครงสร้างเปลี่ยน)
2. แก้ content ใน **ทั้ง** `th/` และ `en/` locale files
3. ทุก content object ต้อง `satisfies` ตาม type ที่กำหนด
